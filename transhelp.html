Hjälp vid transkribering

Den här sidan kan användas för att underlätta vid transkriberingen. Det är ett verktyg för att konvertera ett enklare format till det json-format som vi använder för varje transkribering. Börja med att definiera de olika parterna i samtalet. Detta görs genom att skriva kortkod=Benämning på personen. T.ex: f=Fredrik Lindström eller 1=Pizzeria Zorba. Kortkod kan vara vad som helst men typiskt endast ett tecken. Därefter transkriberas själva dialogen med kortkod:replik. Är man osäker på en replik kan man markera med ett inledande ?. Se rutan nedan för exempel.

f=Fredrik Lindström
1=Parkeringsvakten
1:?Parkeringsvakten ???
f:Ja, är det gatukontoret?
1:Ja.
f:Ja. Hej. Tjena. Fredrik Lindström heter jag.
1:Ja.
f:Jag ska in på systemet, odengatan. Tjacka några lådor kalvados till helgen. Vi ska ha en riktigt röjarskiva vet du.

Använd hakparanteser för att beskriva ljud annat än samtal, t.ex. [skratt] eller [glaskrasch].


Använd sedan följande texten när du bidrar till projektet.
